02. Journey

2023

Started CS Engineering at ESPRIT

Enrolled at ESPRIT in Tunisia — diving into Computer Science fundamentals: algorithms, data structures, and systems programming in C.

2024

Shadows of Liberty — SDL Game + Arduino Controller

Contributed to Shadows of Liberty, a 2D video game built in C with SDL. Alongside it, the team built a physical Arduino game controller — custom hardware interfaced directly with the game.

2024

Tunisia Tourism Web App

Built a full web application to showcase Tunisia — users can explore events, hotels, and tourist destinations across the country. Full browsing and booking-style experience.

2025

Consulting Desktop App — Qt / C++ + Arduino

Co-developed a consulting desktop application in Qt & C++ — manages meetings, certificates, learning paths and career guidance. Paired with an Arduino module that counts people in a room, adjusts lighting accordingly, and handles RFID card scanning for access control.

2025

Enterprise Network Architecture & Monitoring

Designed and deployed a multi-department enterprise network in GNS3 — backbone using DHCP, OSPF, DNS and NAT to interconnect all departments. Led the IT department layer: set up Grafana + Prometheus to monitor live metrics across every connected department in real time.

2026

PPW & Talent Bridge — Active Development

Currently building PPW, a personal AES-256-GCM encrypted password manager in Python, and Talent Bridge, a full-stack Java HR recruitment platform with a JavaFX desktop app and web layer.
