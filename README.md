# Hyujnn Portfolio Site

Personal portfolio site for Mohamed Ben Moussa, published with GitHub Pages. The site presents selected projects, technical focus areas, and GitHub-linked portfolio content in a single-page front-end.

## What the Site Includes

- Single-page portfolio layout with project and profile sections
- Responsive front-end for desktop and mobile viewing
- GitHub-driven project showcase behavior in the client-side JavaScript
- Visual effects such as animated backgrounds, reveal transitions, and lightbox interactions
- Static deployment through GitHub Pages

## Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Repository Layout

```text
index.html   main portfolio page
css/         styling and responsive layout
js/          client-side behavior and GitHub integration
.nojekyll    disables Jekyll processing on GitHub Pages
```

## Local Preview

Open `index.html` directly in a browser, or serve the repository with a lightweight static server during development.

## Deployment

This repository is configured for GitHub Pages-style static hosting. Changes pushed to the published branch update the live portfolio site.
